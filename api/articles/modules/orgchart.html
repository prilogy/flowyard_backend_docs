Оргструктура

Получить оргсхему можно по двум запросам:

  • GET/ api/v1/role/orgchart/with-roles - оргструктура с прикрепленными всеми ролями
  • GET/ api/v1/role/orgchart - только оргструктура

Отрисовка оргсхемы происходит по модели OrgchartItem?[][], которая является матрицой, где уже указано настоящее расположение карточек в схеме. Если элемент равен null - карточки на этом месте нет.

Warning

🆕 Был убран параметр, отвечающий за id компании в роуте. Теперь будет возвращаться модель для текущей компании, т.е. user.currentCompanyId

OrgchartItem

Модель отвечающая за карточку оргструктуры

Свойства

Свойство Тип Домен Описание
roleId int >0 Id функции/роли
parentRoleId int >0 Id родительской функции/роли
connections OrgchartItemConnectionTypes[] Список линий ,которые входят в карточку
🆕 isChildRolesVertical bool Указывает, вертикальны ли дочерние карточки
🆕 childRoleIds int[] Список id дочерних функций/ролей

OrgchartItemConnectionTypes

Перечисление, указывающее на тип входящей линии.

Является enum, и может иметь следующие значения:

  • Top = 0 - линия заходит сверху
  • Left = 1 - линия заходит слева
