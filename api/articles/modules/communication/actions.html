Действия при статусе

Действия обрабатываются при помощи ключей в свойстве baseActionKey модели BoardStatusAction

Список доступных действий можно получить по роуту GET/ {id}/calculated контроллера Issue, либо присоединить их к модели Issue в обычных запросах ресурсов при помощи доп. параметров.

Типы действий

Ключ Название Флоу
withdraw Отозвать Открывается диалог в котором выбирается новый issue.dateDeadline и issue.executorUserId
delete Удалить Открывается диалог с подтверждением удаления issue
copy Скопировать Открывается диалог с копированием issue
🆕 rj Отклонить Открывается диалог с вводом текста(причина). Создать IssueHistory вида:
{ issueId: <id задачи>, comment: { text: <причина> }, actions: [ { key: "rj" } ] }
