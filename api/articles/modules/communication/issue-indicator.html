Индикатор проблемы (IssueIndicator)

Позволяет вывести вспомогательную информацию о задаче, например её нарушения.

Свойства

Свойство Тип Описание
name string? Текст, выводимый внутри индикатора
key string Ключ индикатора. (не требует доп. обработки)
colorSchemeKey string Ключ цветовой схемы, с которой нужно отображать индикатор
🆕 icon string? Ключ иконки, которую нужно отобразить в левой части индикатора если name != null, иначе посередине индикатора
Warning

Индикаторы автоматически крепятся ко всем моделям типа IssueShort в свойстве issueShort.calculated.indicators. Для включения в модель Issue - нужно использовать доп. параметр includeIndicators.
