Отчеты

Note

Все запросы ниже указаны для контроллера Report

Important

Если в запросе отмечено указание дат - они указываются в query параметрах DateFrom(от) & DateTo(до)

Important

Даты указываются в ISO 8601

Коммуникации

По статусам

Запрос - GET/ custom-rule

Указание дат - ✅

Флоу

  1. Из тела ответа берем .items и рендерим список объектов, где в каждом есть также .items, которые реализуют группировку по пользователям. value здесь - счетчик, отображать напротив пользователя, которого брать по userId из .users тела ответа.
  2. Для каждого userId делаем вложенный список из Issue, которые при раскрытии списка получаем с помощью контроллера Issue и фильтрами:
    • ExecutorUserId - id пользователя
    • CustomFilterKey - key объекта в .items
    • CustomFilterDateFrom - дата "от"
    • CustomFilterDateTo - дата "до"

🆕 По нарушениям

Запрос - GET/ violation

Указание дат - ❌

Флоу

  1. Из раздела По статусам - Флоу
  2. Для каждого userId делаем вложенный список из Issue, которые при раскрытии списка получаем с помощью контроллера Issue и фильтрами:
    • WithViolationUserId - id пользователя
    • WithViolationKey - key объекта в .items

🆕 Драйверы

Запрос - GET/ driver

Указание дат - ✅

Запрос возвращает модель DriverReport

DriverReport

DriverReport {
  writesCount integer($int32) // Написали в мессенджеры / Wrote to messenger
  callsCount integer($int32) // Позвонили / Called
  fixedViolations integer($int32) // Устранили нарушений / Fixed violations
  processAvgTimeInMinutes number($double) // Среднее время обработки / Average process time
  overallTimeSavedInHours number($double) // Сэкономлено времени / Saved time
}
