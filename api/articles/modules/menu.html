Меню

Элементы меню приходят с сервера в initialInfo.menuItems в виде типа MenuItemWithChildren[], где сам MenuItemWithChildren:

Свойство Тип Домен Описание
id int >0 Id, по нему происходит сортировка. От меньшего к большему
name string Название
children MenuItemWithChildren[] Вложенные элементы меню, если свойство не пустое - элементы коллекции отображаются под текущим элементом с визуальным отображением вложенности
key string? Ключ элемента меню
isCollapsible bool true - подменю может сворачиваться и разворачиваться (по умолчанию свернуто)
false - подменю отображается в статическом развернутом виде (нет иконки сворачивания и соотв. нельзя свернуть)
IsPending bool true - напротив пункта меню мигает красный индикатор
false - нет индикатора
IsPendingValue string Значение, выводимое в красном индикаторе
Note

🆕 На основе свойства key на фронте привязываются действия, иконки и т.д. к этому элементу меню.
Если ключа нет, то элемент является сугубо разделителем (если Name непустое и не null, также выводится эта строка) , и если свойство children непустое - разделом.

Warning

Вложенность свойства children в уровне глубины не ограничена.
