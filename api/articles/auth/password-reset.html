🆕 Сброс пароля

Флоу

  1. Запрос POST/ api/v1/authentication/password-reset с моделью { email } для получения сообщения на Email с кодом сброса пароля.
    Если пользователь находится в авторизованном состоянии - почту указывать не нужно.
  2. Запрос POST/ api/v1/confirmation для валидации кода, с моделью вида 
    {
  "code": "<код подтверждения>",
  "type": "password_reset"
}
  3. В случае успеха валидации кода, запрос PUT/ api/v1/authentication/password-reset с моделью вида: 
    {
  "confirmationCode": "<код подтверждения>",
  "newPassword": "<новый пароль>"
}
