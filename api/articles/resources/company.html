Компания (Company)

Компания ялвяется основополагающей, на ряду с User, сущностью. Является контекстом рабочего пространства пользователя.

Свойства

Свойство Тип Домен Описание
schedule CompanySchedule[] Size == 7 График работы компании. 7 элементов, по 1 на каждый день недели
timeZoneFromUtc int -12...12 Смещение часового пояса относительно UTC в часах
userOwnerId int >0 Id владельца компании
scheduleIsSameOnWeekdays bool Отвечает за начальное состояние чекбокса "Все будни по одному графику".
Является геттером
weekStart int 0 / 1 Начало недели. Может принимать только 0(Воскресенье) или 1(понедельник)
🆕 imageId int Лого компании. Указывается Id статического файла типа image, загруженного заранее.
