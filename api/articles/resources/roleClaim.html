Клеим функции (RoleClaim)

С помощью этой сущности осуществляется определение разрешений функции.

Свойства

Свойство Тип Домен Описание
Type string Подробнее ниже Тип разрешения
RoleId int >0 Id функции
Note

Одновременно может существовать только одна запись с одиаковой парой { roleId, type }.

Типы разрешений

Актуальные типы, вместе с их локализованными названиями приходят по запросу GET/ api/v1/helper/permission в виде модели Permission.

Warning

У модели Permission есть свойство .children, в котором хранятся вложенные разрешения. Аналогично с моделью MenuItemWithChildren

Если для объекта модели Permission выполняется услвоие .key == null - он является разделителем, а при непустом свойстве .children - разделом.
