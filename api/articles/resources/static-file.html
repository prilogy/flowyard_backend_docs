Статический файл (StaticFile)

Сущность для хранения статических файлов.

Свойства

Свойство Тип Домен Описание
Url string URL расположения файла
FileName string Название файла, с которым он был загружен в систему
Type string Тип файла

Типы файлов

  • file - любой файл, который будет храниться в первоначальном, несжатом виде
  • image - изображение (или медиа контент), к такому типу при загрузке применяется сжатие. При загрузке не медиа файлов с таким типом сервер будет выдавать ошибку, т.к. не сможет сжать его.

Загрузка на сервер

Производится по пути POST/ range контроллера StaticFile.

Тип тела HTTP запроса - form-data, файлы крепятся под ключ files. В query параметре type указывается тип загружаемых файлов. Одновременно можно загружать только один тип файлов.

Флоу использования статических файлов в других сущностях

Сначала загружается статический файл с нужным типом, далее в нужной сущности указывается Id этого загруженного файла в конкретном свойстве.
