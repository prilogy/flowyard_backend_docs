Ресурсы / Введение

Ресурс представляет доменную сущность системы. Ресурс(-ы) можно получать, изменять и удалять.

Получение списка

Эндпоинт по пути GET/ api/{v}/{resource} имеет несколько особенностей

Пагинация

Имеются следующие параметры

Название Значение по умолчанию Домен Описание
page 1 >0 Номер страницы
pageSize 10 >0 Размер страницы
orderByDate desc asc / desc Порядок выдачи.
asc возвращает объекты с id 1, 2, 3 (по порядку)
desc возвращает объекты с id 3, 2, 1 (в обратном порядке)

Фильтрация

Каждый такой эндпоинт может иметь свой собственный фильтр, который передается через query параметры. Подробная спецификация фильтров находится в Swagger.
