Требование доказательства проблемы (IssueProofRequirement, ранее IssueProof)

С помощью этой сущности задаются правила проверки доказательства для проблемы.

Если свойство issue.proof с этим типом равно null - доказательство не требуется.

Свойства

Свойство Тип Домен Req. Описание
text string ✅ Текст к требованию доказательства
rules IssueProofRequirementRule[]? size > 0 ✅ Правила к требованию доказательства

Правило (IssueProofRequirementRule, ранее IssueProofRule)

С помощью этой сущности конкретно задаются требования к комметарию, чтобы он считался доказательством.

Свойства

Свойство Тип Домен Req. Описание
value int >0 ✅ Числовое значение, связанное с правилом
key string ✅ Ключ типа правила
Ключи типов правил
  • require.text_min_length - минимальное число символов текста комментария
  • require.images_min_count - минимальное число изображений, прикрепленных к комментарию
