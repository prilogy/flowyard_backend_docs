Образовательный элемент (EducationItem)

Свойства

Свойство Тип Домен Описание
name string? Название
videoId string? Id видео (YouTube)
children EducationItem[]? Дочерние элементы.
Если этот массив не null и не пуст - отображать как раздел, где name - его название
isAvailable bool Доступность элемента. Если true - доступен.
Note

🆕 Данные можно получить из запроса GET / контроллера Education.

Warning

Документация по сохранению прогресса обучения - в разработке...

🆕 Прогресс обучения

Хранится в user.educationHistory (возвращается только для авторизованного пользователя).

Сохранение прогресса

Чтобы сохранить прогресс, нужно отправить запрос POST / контроллера Education, где key тела запроса - ключ обучающего элемента. Подобности по запросу - в Swagger.

Ключи обучающих элементов

// Основы
public const string Intro = "ed.b.i"; // Вводное видео
public const string Overview = "ed.b.o"; // Обзор платформы
public const string Settings = "ed.b.s"; // Настройка личного профиля
    

// Оргструктура
public const string Intro = "ed.o.i"; // Вводное видео
public const string SettingUpOrgchart = "ed.o.s"; // Технические настройки
    
    
// Коммуникации
public const string Intro = "ed.c.i"; // Вводное видео
public const string WorkingWithIssues = "ed.c.wwi"; // Работа с задачами
public const string RecurringIssues = "ed.c.ri"; // Повторяющиеся задачи
public const string SettingUpRecurringIssues = "ed.c.s"; // Настройки повторяющихся задач
