Конфиг исполнителей повторяющейся проблемы (RecurringIssueExecutorConfig)

Задает правила для подбора исполнителей на повторяющуюся проблему.

Свойства

Свойство Тип Домен Req. Описание
type string ✅ Тип конфига
roleId int? Id функции, если этого требует тип конфига
isExcluded bool ✅ Реализует функционал "кроме" если true

Типы конфига

  • all - все
  • all_admins - все администраторы
  • role_executors - исполнители функции *
  • role_admins - руководители функции *

* - обязательно указание свойства roleId
