Повторяющаяся проблема (RecurringIssue)

Проблема которая автоматически создается (повторяется) по заданным правилам.

Свойства

Note

Чтобы таблица умещалась без скролла, названия некоторых свойств перенесены на вторую строку двумя точками

Свойство Тип Домен Req. Описание
name string ✅ Название
description string ✅ Описание
awaitedResult string >0 ✅ Результат

createdOnRoleId int >0 ✅ Id функции, на которой создана сущность
createdByUserId int >0 Id пользователя - создателя

* - формат months.days.hours:minutes, все параметры обязательны, даже если равны 0

** - обязательно указание только одного из этих свойств

Типы правил подбора отправителя

  • nearest_admin - ближайший руководитель
  • executor_of_role - исполнитель функции (обязательно указание свойства senderRoleId)
  • admin_of_role - руководитель функции (обязательно указание свойства senderRoleId)

Типы правил таймзоны

  • of_executor - таймзона исполнителя
  • of_creator - таймзона отправителя
  • of_company - таймзона компании
